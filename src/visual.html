📋 Análise Comparativa: Desempenho Esperado vs. Obtido

Algoritmo Complexidade Esperada Comportamento Observado Correspondência
Código 1 O(n) — Linear Operações = n/2. Ao dobrar n, operações dobram. Tempo cresce linearmente. ✅ Confirmado
Código 2 O(n log n) Operações ≈ 2n. O loop externo executa log₂(n) vezes e o interno soma ≈ 2n iterações. Tempo cresce de forma quase linear. ✅ Confirmado
Código 3 O(n²) — Quadrático Operações ≈ n²/2. Ao dobrar n, operações quadruplicam (~4x). Tempo segue padrão quadrático. ✅ Confirmado
Código 4 O(2ⁿ) — Exponencial Operações crescem exponencialmente. De n=24 para n=48, operações saltam de ~93 mil para ~9,6 bilhões. Tempo explode. ✅ Confirmado
